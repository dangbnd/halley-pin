# Photo Gallery (Pinterest UI + DB)

## Run (local)
```bash
npm i
npx prisma generate
npx prisma migrate dev
npm run dev
```

- Gallery: http://localhost:3000/gallery
- Admin login: http://localhost:3000/admin

## Notes
- Guest UI tối giản (ảnh + title + category)
- Admin đăng nhập mới thấy Upload/Sync + xoá + sửa “Thông tin bánh”
